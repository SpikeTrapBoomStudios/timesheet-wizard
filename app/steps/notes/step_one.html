Source Timesheet File Requirements

The source timesheet file is an Excel spreadsheet that contains the raw timesheet data used by this application. For the application to work correctly, the spreadsheet must follow a strict structure. Please review the requirements below carefully.

General Rules

  • The file must be an Excel (.xlsx) file
  • The spreadsheet may contain any number of rows

Column Requirements

  • All non-date columns must have the exact same names and order every time
  • Date columns are the only exception
    • You may name date columns however you like
    • Date columns must remain in the same order

Changing the order, removing columns, or renaming non-date columns will cause the file to be rejected or processed incorrectly.
