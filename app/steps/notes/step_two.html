Select Source Sheet and Start Date

Instructions

  • Select the sheet from the source Excel file that contains the raw timesheet data.
  • For the start date, use the calendar to select the first date the timesheets begin on.
  • The wizard will automatically generate a complete timesheet period by incrementing dates forward from the selected start date.

Timesheet Period

Each generated timesheet will span 14 consecutive days, ending 13 days after the selected start date.
